## 1. 实施概述

### 1.1 目标

创建小遥搜索 MCP 能力的 **Agent Skill**，使 Claude Code、VS Code、Cursor、OpenCode 等 AI 助手能够通过 MCP 协议发现和使用小遥搜索的本地文件搜索能力。

### 1.2 核心决策点（已确定）

| 决策项 | 决策结果 | 状态 |
|--------|---------|------|
| **Skill 名称** | `xiaoyao-search` | ✅ 已确定 |
| **目录位置** | `.claude/skills/xiaoyao-search/` | ✅ 已确定 |
| **协议标准** | MCP 协议（通用标准） | ✅ 已确定 |
| **工具范围** | 5 个搜索工具 | ✅ 已确定 |
| **配置方式** | YAML frontmatter | ✅ 已确定 |

### 1.3 设计原则

- **标准化**：遵循 Claude Agent Skills 规范格式
- **跨平台**：支持多种 AI 助手（MCP 协议通用）
- **开箱即用**：AI 助手可自动发现可用工具
- **完整文档**：包含使用示例和故障排查指南
- **最小侵入**：纯文档类 Skill，不修改现有代码

## 3. 实施步骤

### 3.1 第一阶段：创建 Skill 目录和主文件

#### 步骤 1：创建 Skill 目录

```bash
mkdir -p .claude/skills/xiaoyao-search/rules
```

#### 步骤 2：编写 SKILL.md 主文件

**文件**: `.claude/skills/xiaoyao-search/SKILL.md`

包含内容：
- YAML frontmatter（name, description, metadata）
- When to use（使用场景）
- Capability（5 个工具定义）
- Usage（使用说明）
- Examples（调用示例）
- Configuration（配置说明）
- Troubleshooting（故障排查）

**预计时间**：1 小时

### 3.2 第二阶段：编写规则文件

#### 步骤 3：编写 tools.md 工具详细定义

**文件**: `.claude/skills/xiaoyao-search/rules/tools.md`

包含内容：
- 每个工具的完整参数定义
- 参数类型、默认值、有效范围
- 返回值格式说明
- 错误处理说明

**预计时间**：1 小时

#### 步骤 4：编写 connection.md 连接配置指南

**文件**: `.claude/skills/xiaoyao-search/rules/connection.md`

包含内容：
- MCP SSE 端点说明
- 各 AI 助手配置示例（Claude Code、VS Code、Cursor、OpenCode）
- 环境变量配置
- 连接测试方法

**预计时间**：0.5 小时

#### 步骤 5：编写 examples.md 使用示例

**文件**: `.claude/skills/xiaoyao-search/rules/examples.md`

包含内容：
- 5 种搜索场景的完整示例
- 常见使用模式
- 最佳实践建议

**预计时间**：1 小时
