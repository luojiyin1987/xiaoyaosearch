## 设计原则：FastAPI 集成 + SSE 端点 + 适配器模式

本特性采用 **FastAPI 集成方案**，将 MCP 服务器集成到现有 FastAPI 进程中：
- **集成架构**：MCP 服务器作为 FastAPI 的一部分，共享 AI 模型和搜索服务
- **SSE 传输**：通过 SSE 端点与 Claude Desktop 通信
- **适配器模式**：通过 SearchAdapter 复用现有搜索服务
- **向后兼容**：不影响现有功能和 API
- **功能开关**：可通过环境变量启用/禁用

if settings.mcp_sse_enabled:
    mcp_server = create_mcp_server()
    app.state.mcp_server = mcp_server

@app.get("/mcp/sse")
