# 小遥搜索 XiaoyaoSearch 产品原型 v2.0

## 1. 设计理念

### 1.1 设计原则
- **极简而有张力**：去除冗余元素，但保持视觉冲击力
- **智能无感体验**：AI驱动的交互如呼吸般自然
- **本地数据堡垒**：用户数据完全掌控，零云端依赖
- **多模态无缝融合**：语音、文本、图像输入的流体式切换

### 1.2 设计风格
- **现代科技主义 + 有机自然主义**：硬朗几何与流动曲线的对比融合
- **深邃光影渐变系统**：多层次光效和动态色彩变化
- **沉浸式布局**：最大化内容展示，最小化界面干扰
- **微交互动画**：细腻的状态变化和情感化反馈

3. **智能降噪处理** → 自动过滤背景噪音，增强识别准确度
4. **语义理解增强** → AI自动提取关键词和意图

4. **渐进式搜索** → 输入过程中实时显示搜索建议

**🖼️ 图片输入交互2.0：**
1. **拖拽上传** → 支持直接拖拽图片到搜索框

#### 3.2.2 MCP服务器配置 🚧 规划中

**配置说明**：
MCP 服务器是纯后端特性，用户界面为 **Claude Desktop**（第三方应用），本配置主要描述配置方式。

**Claude Desktop 配置文件路径**：
- Windows: `%APPDATA%\Claude\claude_desktop_config.json`
- macOS: `~/Library/Application Support/Claude/claude_desktop_config.json`
- Linux: `~/.config/Claude/claude_desktop_config.json`

**配置示例**：
```json
{
  "mcpServers": {
    "xiaoyao-search": {
      "url": "http://127.0.0.1:8000/mcp/sse",
      "disabled": false
    }
  }
}
```

**快速配置流程**：
1. 启动小遥搜索后端服务：`cd backend && python main.py`
2. 确认服务正常运行：访问 http://127.0.0.1:8000/docs
3. 编辑 Claude Desktop 配置文件，添加上述配置
4. 重启 Claude Desktop
5. 在 Claude 对话中测试：`"帮我搜索关于人工智能的文档"`

**可用的 MCP 搜索工具**：
- `semantic_search` - 语义搜索（基于 BGE-M3）
- `fulltext_search` - 全文搜索（基于 Whoosh）
- `voice_search` - 语音搜索（基于 FasterWhisper）
- `image_search` - 图像搜索（基于 CN-CLIP）
- `hybrid_search` - 混合搜索（语义+全文）

**详见**：[MCP服务器支持原型设计](特性开发/mcp/mcp-02-原型.md)

const useDebounce = (fn: Function, delay: number) => {
  let timeoutId: NodeJS.Timeout
  return (...args: any[]) => {
    clearTimeout(timeoutId)
    timeoutId = setTimeout(() => fn.apply(null, args), delay)
  }
}

// 虚拟滚动优化
const useVirtualScroll = (items: any[], itemHeight: number) => {
  const scrollTop = ref(0)
  const containerHeight = ref(600)

  const visibleItems = computed(() => {
    const startIndex = Math.floor(scrollTop.value / itemHeight)
    const endIndex = Math.min(
      startIndex + Math.ceil(containerHeight.value / itemHeight) + 1,
      items.length
    )

    return items.slice(startIndex, endIndex).map((item, index) => ({
      ...item,
      index: startIndex + index
    }))
  })

  return { visibleItems, scrollTop, containerHeight }
}
