## 🎯 当前开发任务

### 🔥 飞书数据源插件（第一优先级）
- **进度**: 20% 开发中
- **开始时间**: 2026-03-31
- **预计工期**: 1周（约40小时）
- **文档**: [PRD](docs/特性开发/plugins+feishu/plugins+feishu-01-prd.md)

**核心功能**：
- 为小遥搜索添加飞书文档数据源识别能力
- 解析飞书导出文档末尾的元数据块
- 提取原文链接和更新时间
- 搜索结果显示"飞书"来源标识
- 支持跳转到飞书原文

**技术实现**：
- **插件位置**：`backend/data/plugins/datasource/feishu/plugin.py`
- **接口定义**：继承 `DataSourcePlugin` ABC基类
- **元数据解析**：正则表达式匹配飞书元数据格式
- **文件扫描**：集成到 `FileScanner` 中自动检测
- **数据存储**：使用已有字段（source_type、source_url）

**开发步骤**:
1. 🚧 创建飞书插件目录结构
2. 🚧 实现 FeishuDataSource 插件类
3. 🚧 实现元数据解析逻辑（正则表达式）
4. 🚧 修改 FileScanner 集成飞书检测
5. 🚧 编写单元测试
6. 🚧 功能测试与验证
7. 🚧 文档更新

**核心功能**：

- 本地/云端互斥设计，切换时自动处理索引重建

**开发步骤**:
1. 🚧 后端服务实现：创建OpenAIEmbeddingService服务类
2. 🚧 后端API修改：支持embedding+cloud互斥配置
