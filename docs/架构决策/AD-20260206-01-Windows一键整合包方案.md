# 架构决策 - Windows一键整合包方案

**决策日期**：2026年02月06日

## 背景

小遥搜索作为本地桌面应用，需要为普通用户提供友好的安装和使用体验。

### 原方案回顾（AD-20260203-08）

| 方案 | 问题 |
|------|------|
| **PyInstaller EXE打包** | 打包时间长、体积大、易被杀毒误报、依赖更新困难 |
| **Electron Builder集成** | 前后端打包流程复杂、更新维护成本高 |
| **多平台支持** | macOS/Linux 用户占比低，维护成本高 |

### 新方案决策

经过重新评估，确定采用**Windows一键整合包方案**：

| 决策项 | 选择 | 理由 |
|--------|------|------|
| **前端打包** | 不打包，保留源码 | 避免Electron打包复杂性、便于调试 |
| **后端打包** | 不打包，保留源码 | 避免杀毒误报、便于调试、依赖更新简单 |
| **运行时分发** | 预包含安装包 | ffmpeg、ollama、node、python 安装包预置 |
| **环境配置** | 自动安装脚本 | 一键配置所有运行时环境 |
| **平台支持** | 仅Windows | 用户群体集中、降低维护成本 |
| **启动方式** | 运行脚本管理 | 自动启动前后端进程、统一管理 |

## 技术实现

### 环境配置脚本

**scripts/setup-env.bat - 一键环境配置：**

```batch
@echo off
title 小遥搜索 - 环境配置
chcp 65001 >nul

echo ========================================
echo   小遥搜索 环境配置向导
echo ========================================
echo.

:: 检查管理员权限
net session >nul 2>&1
if %errorLevel% neq 0 (
    echo [警告] 建议以管理员身份运行此脚本
    echo.
)

:: 1. 检测环境
echo [1/5] 检测现有环境...
call "%~dp0check-env.bat"
echo.

:: 2. 安装 Python
echo [2/5] 安装 Python 3.10.11...
python --version >nul 2>&1
if %errorLevel% neq 0 (
    echo 正在安装 Python...
    installers\python-3.10.11-amd64.exe /quiet InstallAllUsers=0 PrependPath=1 Include_test=0
    echo Python 安装完成
) else (
    echo Python 已安装，跳过
)
echo.

:: 3. 安装 Node.js
echo [3/5] 安装 Node.js...
node --version >nul 2>&1
if %errorLevel% neq 0 (
    echo 正在安装 Node.js...
    msiexec /i installers\node-v21.x-x64.msi /qn
    echo Node.js 安装完成
    :: 刷新环境变量
    refreshenv >nul 2>&1
) else (
    echo Node.js 已安装，跳过
)
echo.

:: 4. 安装 FFmpeg
echo [4/5] 安装 FFmpeg...
ffmpeg -version >nul 2>&1
if %errorLevel% neq 0 (
    echo 正在安装 FFmpeg...
    powershell -Command "Expand-Archive -Path installers\ffmpeg-essentials.zip -DestinationPath C:\ffmpeg -Force"
    setx PATH "%PATH%;C:\ffmpeg\bin" /M
    echo FFmpeg 安装完成
) else (
    echo FFmpeg 已安装，跳过
)
echo.

:: 5. 安装 Ollama
echo [5/5] 安装 Ollama...
ollama --version >nul 2>&1
if %errorLevel% neq 0 (
    echo 正在安装 Ollama...
    powershell -Command "Expand-Archive -Path installers\ollama-windows-amd64.zip -DestinationPath %LOCALAPPDATA%\Ollama -Force"
    setx PATH "%PATH%;%LOCALAPPDATA%\Ollama" /M
    echo Ollama 安装完成
) else (
    echo Ollama 已安装，跳过
)
echo.

:: 6. 安装 Python 依赖
echo 安装后端 Python 依赖...
cd backend
python -m venv .venv
call .venv\Scripts\activate.bat
pip install -r requirements.txt -i https://pypi.tuna.tsinghua.edu.cn/simple
cd ..
echo.

:: 7. 安装 Node 依赖
echo 安装前端 Node 依赖...
cd frontend
call npm install
cd ..
echo.

:: 8. 生成配置文件
if not exist "config.yaml" (
    echo 生成默认配置文件...
    copy config.yaml.example config.yaml
)
echo.

echo ========================================
echo   环境配置完成！
echo ========================================
echo.
echo 下一步：双击 start.bat 启动应用
pause
```

### 启动管理脚本

**scripts/start.bat - 一键启动脚本：**

```batch
@echo off
title 小遥搜索
chcp 65001 >nul

echo ========================================
echo   小遥搜索 启动中...
echo ========================================
echo.

:: 检测环境
call "%~dp0check-env.bat"

:: 检查是否已配置环境
if not exist "backend\.venv\Scripts\python.exe" (
    echo.
    echo [错误] 环境未配置，请先运行 setup-env.bat
    pause
    exit /b 1
)

if not exist "frontend\node_modules\" (
    echo.
    echo [错误] 前端依赖未安装，请先运行 setup-env.bat
    pause
    exit /b 1
)

:: 创建必要目录
if not exist "data\database" mkdir data\database
if not exist "data\indexes" mkdir data\indexes
if not exist "data\configs" mkdir data\configs
if not exist "data\logs" mkdir data\logs
if not exist "data\uploads" mkdir data\uploads

:: 启动后端
echo.
echo [*] 启动后端服务...
cd backend
start "小遥搜索-后端" /MIN cmd /c ".venv\Scripts\python.exe main.py"
cd ..

:: 等待后端启动
echo [*] 等待后端服务启动...
timeout /t 5 /nobreak >nul

:: 启动前端
echo [*] 启动前端界面...
cd frontend
start "小遥搜索-前端" cmd /c "npm run dev"
cd ..

echo.
echo ========================================
echo   小遥搜索已启动！
echo ========================================
echo.
echo 后端地址: http://127.0.0.1:8000
echo 前端地址: http://127.0.0.1:5173
echo.
echo 提示：关闭此窗口不会停止服务
echo 请使用 stop.bat 停止所有服务
echo.
pause
```

**scripts/stop.bat - 一键停止脚本：**

```batch

title 停止小遥搜索
chcp 65001 >nul

echo ========================================

echo [*] 停止前端界面...
taskkill /FI "WINDOWTITLE eq 小遥搜索-前端*" /F >nul 2>&1

:: 停止 Ollama（可选）

taskkill /IM ollama.exe /F >nul 2>&1

echo.
echo ========================================
echo   小遥搜索已停止
echo ========================================
pause
```

### 进程管理优化

使用 **PID 文件管理** 进程：

```batch
:: start.bat 改进版
:: 启动后端并记录 PID
cd backend
start "小遥搜索-后端" /MIN cmd /c ".venv\Scripts\python.exe main.py"
for /f "tokens=2" %%i in ('tasklist /FI "WINDOWTITLE eq 小遥搜索-后端*" /NH /FO CSV') do set BACKEND_PID=%%i
echo %BACKEND_PID% > ..\data\backend.pid
cd ..

:: stop.bat 改进版
:: 读取 PID 并关闭进程
if exist "data\backend.pid" (
    set /p BACKEND_PID=<data\backend.pid
    taskkill /PID %BACKEND_PID% /F >nul 2>&1
    del data\backend.pid
)
```

mkdir "%BUILD_DIR%\frontend"

:: 复制后端源码
echo [*] 复制后端源码...
xcopy /E /I /Y "backend\app" "%BUILD_DIR%\backend\app\"

echo [*] 复制脚本...

:: 生成配置模板
echo [*] 生成配置模板...
echo # 小遥搜索配置文件 > "%BUILD_DIR%\config.yaml.example"
echo app: >> "%BUILD_DIR%\config.yaml.example"
echo   language: zh_CN >> "%BUILD_DIR%\config.yaml.example"
echo   theme: auto >> "%BUILD_DIR%\config.yaml.example"

:: 下载运行时安装包（如果不存在）
echo.
echo [*] 检查运行时安装包...
echo.
echo 提示：以下安装包需要手动下载到 installers 目录
echo.
echo 1. Python 3.10.11:
echo    https://www.python.org/ftp/python/3.10.11/python-3.10.11-amd64.exe
echo    保存为: installers\python-3.10.11-amd64.exe
echo.
echo 2. Node.js 21.x LTS:
echo    https://nodejs.org/dist/latest-v21.x/node-v21.x-x64.msi
echo    保存为: installers\node-v21.x-x64.msi
echo.
echo 3. FFmpeg Essentials:
echo    https://www.gyan.dev/ffmpeg/builds/ffmpeg-release-essentials.zip
echo    保存为: installers\ffmpeg-essentials.zip
echo.
echo 4. Ollama:
echo    https://ollama.ai/download/windows
echo    保存为: installers\ollama-windows-amd64.zip

    echo [警告] 缺少 Python 安装包
    set MISSING=1

if not exist "%BUILD_DIR%\installers\node-v21.x-x64.msi" (
    echo [警告] 缺少 Node.js 安装包

if not exist "%BUILD_DIR%\installers\ffmpeg-essentials.zip" (

)
if not exist "%BUILD_DIR%\installers\ollama-windows-amd64.zip" (
    echo [警告] 缺少 Ollama 安装包

## 分发策略

### 平台支持

| 平台 | 支持方式 | 交付物 |
|------|---------|--------|
| **Windows 10/11** | 一键整合包 | `XiaoyaoSearch-Windows-v1.0.0.zip` |
| **macOS** | 暂不支持 | - |
| **Linux** | 暂不支持 | - |

### 用户使用流程

```
1. 下载整合包压缩包（约800MB）
   XiaoyaoSearch-Windows-v1.0.0.zip

2. 解压到任意目录（建议非系统盘）
   D:\MyApps\XiaoyaoSearch-Windows-v1.0.0\

3. 右键以管理员身份运行 setup-env.bat
   - 自动检测已安装的运行时
   - 自动安装缺失的运行时
   - 自动安装 Python 和 Node 依赖
   - 首次配置约需 5-10 分钟

4. （首次使用）下载AI模型
   - 参考 MODELS.md 下载指南
   - 解压到 data/models/ 目录

5. 双击 start.bat 启动应用
   - 自动启动后端和前端服务
   - 自动打开浏览器访问 http://127.0.0.1:5173

6. 开始使用，选择任意文件夹进行索引

7. 使用完毕双击 stop.bat 停止服务
```

## 影响分析

### 正面影响

1. **零门槛部署**：
   - 双击 setup-env.bat 即可完成环境配置
   - 自动处理所有依赖安装

2. **开发友好**：
   - 源码完全保留，便于调试
   - 依赖更新无需重新打包
   - 与开发环境一致

3. **透明可控**：
   - 用户可以看到所有运行过程
   - 可以手动干预和调整配置
   - 不会被杀毒软件误报

4. **维护成本低**：
   - 仅支持 Windows，减少负担
   - 不需要复杂的打包流程
   - 更新只需替换源码文件

### 需要注意

1. **首次配置时间**：
   - 环境配置需要 5-10 分钟
   - 需要友好的进度提示

2. **网络依赖**：
   - 依赖安装需要网络连接
   - 已预置安装包减少外网依赖

3. **磁盘空间**：
   - 整合包约 800MB
   - 安装后约 2GB（含运行时和依赖）

4. **管理员权限**：
   - 部分运行时安装需要管理员权限
   - 脚本会提示用户

### 与原方案对比

| 对比项 | 原方案（EXE打包） | 新方案（整合包） |
|--------|------------------|-----------------|
| **打包时间** | 30-60分钟 | 5-10分钟 |
| **杀毒误报** | 容易误报 | 无误报 |
| **调试难度** | 高（需反编译） | 低（源码可见） |
| **依赖更新** | 需重新打包 | 直接更新 |
| **用户门槛** | 双击运行 | 双击配置后运行 |
| **首次启动** | 即时启动 | 需配置环境 |
| **维护成本** | 高 | 低 |
| **透明度** | 黑盒 | 完全透明 |
