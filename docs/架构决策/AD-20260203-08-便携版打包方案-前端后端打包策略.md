## 背景

小遥搜索作为本地桌面应用，需要为普通用户提供友好的安装和使用体验。

### 问题分析

| 方案 | 问题 |
|------|------|
| **本地Python环境** | 普通用户需要配置Python环境，门槛高 |
| **Docker部署** | 文件访问灵活性受限，不适合桌面应用 |
| **传统安装程序** | 写注册表，不够便携，卸载复杂 |

### 用户决策

经过讨论，最终确定采用**便携版（Portable）打包方案**：

| 决策项 | 选择 |
|--------|------|
| **模型文件分发** | 用户手动下载，放到 data 目录指定位置 |
| **配置文件管理** | 应用目录内（便携性好，用户易找到） |
| **安装程序** | 纯便携版（解压即用，不写注册表） |
| **macOS代码签名** | 跳过（用户首次运行需要右键打开） |

## 解决方案

### 便携版架构设计

```
┌─────────────────────────────────────────────────────────────────┐
│  小遥搜索 便携版目录结构                                        │
├─────────────────────────────────────────────────────────────────┤
│                                                                  │
│  XiaoyaoSearch-Portable/                                        │
│  ├── XiaoyaoSearch.exe          # 主程序（Windows）             │
│  ├── resources/                                                 │
│  │   ├── backend/               # 后端可执行文件                 │
│  │   │   └── xiaoyao-backend.exe  # PyInstaller打包            │
│  │   └── app.asar              # Electron前端资源               │
│  ├── data/                      # 数据目录（首次运行创建）        │
│  │   ├── database/              # SQLite数据库                  │
│  │   ├── indexes/               # 索引文件                       │
│  │   ├── configs/               # 配置文件                       │
│  │   ├── logs/                  # 日志文件                       │
│  │   ├── models/                # AI模型（用户手动下载）         │
│  │   └── uploads/               # 上传文件临时目录               │
│  ├── config.yaml                # 应用配置文件（首次运行创建）    │
│  ├── README.txt                 # 使用说明                       │
│  └── MODELS.md                  # 模型下载指南                   │
│                                                                  │
└─────────────────────────────────────────────────────────────────┘
```

### 技术实现

#### 后端打包

使用 PyInstaller 将 Python 后端打包成独立可执行文件：

```bash
# Windows 打包
pyinstaller --onefile \
  --name xiaoyao-backend \
  --add-data "app:app" \
  --hidden-import fastapi \
  --hidden-import uvicorn \
  --hidden-import sentence_transformers \
  --hidden-import transformers \
  main.py
```

**输出：** `xiaoyao-backend.exe` (约100-150MB)

#### 前端打包

使用 electron-builder 打包 Electron 应用：

```javascript
// electron-builder 配置
{
  "appId": "com.xiaoyaosearch.app",
  "productName": "XiaoyaoSearch",
  "directories": {
    "output": "dist"
  },
  "files": [
    "frontend/dist/**/*",
    "resources/**/*"
  ],
  "win": {
    "target": "portable",
    "artifactName": "XiaoyaoSearch-Portable-${version}.exe"
  },
  "mac": {
    "target": "dir",
    "artifactName": "XiaoyaoSearch-Portable-${version}"
  }
}
```

#### 进程管理

Electron 主进程负责启动和管理后端进程：

```typescript
// 启动后端进程
const backendPath = path.join(process.resourcesPath, 'backend', 'xiaoyao-backend.exe');
const backendProcess = spawn(backendPath, [], {
  cwd: app.getPath('userData')
});

// 应用退出时关闭后端
app.on('before-quit', () => {
  backendProcess.kill();
});
```

## 分发策略

### 平台支持

| 平台 | 支持方式 | 交付物 |
|------|---------|--------|
| **Windows** | 便携版 | `XiaoyaoSearch-Portable-v1.0.0.zip` |
| **macOS** | 便携版 | `XiaoyaoSearch-Portable-v1.0.0.dmg` |
| **Linux** | 源码安装 | 文档指引 |

### 用户使用流程

```
1. 下载便携版压缩包（约200MB）

2. 解压到任意目录
   D:\MyApps\XiaoyaoSearch-Portable\

3. （首次使用）下载AI模型
   - 参考 MODELS.md 下载指南
   - 解压到 data/models/ 目录

4. 双击运行 XiaoyaoSearch.exe

5. 开始使用，选择任意文件夹进行索引
```

### 模型文件管理

**模型下载指南（MODELS.md）：**

| 模型 | 用途 | 大小 | 下载链接 |
|------|------|------|----------|
| BGE-M3 | 文本嵌入 | ~2GB | HuggingFace |
| CN-CLIP | 图像理解 | ~500MB | HuggingFace |
| Faster-Whisper | 语音识别 | ~150MB | HuggingFace |

**用户操作：**
```bash
# 下载模型后解压到指定目录
data/models/
├── bge-m3/
├── cn-clip/
└── faster-whisper/
```

## 配置管理

### 配置文件结构

**config.yaml**（应用目录内）：

```yaml
# 应用配置
app:
  language: zh_CN  # 界面语言
  theme: auto      # 主题：auto/light/dark

# 后端配置
backend:
  host: 127.0.0.1
  port: 8000

# 模型配置
models:
  bge_m3_path: ./data/models/bge-m3
  clip_path: ./data/models/cn-clip
  whisper_path: ./data/models/faster-whisper

# 索引配置
index:
  chunk_size: 500
  chunk_overlap: 50
```

### 首次启动流程

```
1. 检测 config.yaml 是否存在
    ├─ 存在 → 加载配置
    └─ 不存在 → 生成默认配置

2. 检测模型文件是否存在
    ├─ 存在 → 直接启动
    └─ 不存在 → 显示模型下载指引

3. 创建必要的数据目录
   - data/database/
   - data/indexes/
   - data/logs/
   - data/uploads/
```

## 影响

### 正面影响

1. **用户体验最佳**：解压即用，无需安装
2. **完全便携**：可放在U盘、任意目录运行
3. **灵活访问**：保持本地应用的文件访问灵活性
4. **零依赖**：用户无需安装 Python、Docker 等

### 需要注意

1. **打包体积**：便携版约 200MB（不含模型）
2. **模型下载**：用户需手动下载模型文件
3. **macOS 门槛**：首次运行需右键打开（未签名）
4. **病毒查杀**：PyInstaller 打包可能被误报
