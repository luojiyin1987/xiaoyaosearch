## CPU 与 GPU 版本

小遥搜索提供两个版本的后端可执行文件：

| 特性 | CPU版本 | GPU版本 |
|------|---------|---------|
| **适用用户** | 大多数普通用户 | 有NVIDIA显卡的用户 |
| **PyTorch** | torch (CPU) | torch (CUDA 12.1) |
| **Faiss** | faiss-cpu | faiss-gpu |
| **打包大小** | ~150MB | ~200MB |
| **推理速度** | 基准 | 5-10倍加速 |
| **输出文件** | `xiaoyao-backend-cpu.exe` | `xiaoyao-backend-gpu.exe` |

## 快速开始

### CPU版本打包（推荐）

```bash
cd backend

# 确保安装了CPU版本的依赖
pip install torch torchvision torchaudio --index-url https://download.pytorch.org/whl/cpu
pip install faiss-cpu

# 打包
pyinstaller xiaoyao_backend_cpu.spec
```

### GPU版本打包

```bash
cd backend

# 确保安装了GPU版本的依赖
pip install torch torchvision torchaudio --index-url https://download.pytorch.org/whl/cu121
pip install faiss-gpu

# 打包
pyinstaller xiaoyao_backend_gpu.spec
```

## 打包流程

### 1. 准备环境

```bash
# 安装 PyInstaller
pip install pyinstaller

# 安装项目依赖
cd backend
pip install -r requirements.txt
```

### 2. 选择版本

**CPU版本（大多数用户）：**
```bash
# 安装CPU依赖
pip install torch torchvision torchaudio --index-url https://download.pytorch.org/whl/cpu
pip install faiss-cpu

# 打包
pyinstaller xiaoyao_backend_cpu.spec
```

**GPU版本（有NVIDIA显卡）：**
```bash
# 安装GPU依赖
pip install torch torchvision torchaudio --index-url https://download.pytorch.org/whl/cu121
pip install faiss-gpu

# 打包
pyinstaller xiaoyao_backend_gpu.spec
```

### 3. 验证输出

```bash
# 检查输出文件
dir dist\xiaoyao-backend-cpu.exe    # CPU版本
dir dist\xiaoyao-backend-gpu.exe    # GPU版本

# 测试运行
dist\xiaoyao-backend-cpu.exe
```

## 完整打包脚本

创建 `scripts/build-backend.bat`：

```batch
@echo off
echo ========================================
echo   小遥搜索后端打包脚本
echo ========================================

echo.
echo 请选择打包版本:
echo   1. CPU 版本（推荐大多数用户）
echo   2. GPU 版本（需要NVIDIA显卡）
echo.
set /p version="请输入选择 (1 或 2): "

cd backend

if "%version%"=="2" (
    echo.
    echo [*] 安装GPU版本依赖...
    pip install torch torchvision torchaudio --index-url https://download.pytorch.org/whl/cu121
    pip install faiss-gpu
    echo.
    echo [*] 打包GPU版本...
    pyinstaller xiaoyao_backend_gpu.spec
) else (
    echo.
    echo [*] 安装CPU版本依赖...
    pip install torch torchvision torchaudio --index-url https://download.pytorch.org/whl/cpu
    pip install faiss-cpu
    echo.
    echo [*] 打包CPU版本...
    pyinstaller xiaoyao_backend_cpu.spec
)

echo.
echo ========================================
echo   打包完成！
echo ========================================
echo.
echo 输出文件: backend\dist\
pause
```

## 常见问题

### Q1: 打包卡在 "Performing binary vs. data reclassification"

**解决方案：**
1. 已在 spec 文件中禁用 UPX 压缩 (`upx=False`)
2. 等待时间可能需要 10-30 分钟
3. 如果超过 30 分钟，按 Ctrl+C 中止并重试

### Q2: GPU版本打包后无法运行

**可能原因：**
- 目标机器没有 NVIDIA 显卡
- 没有安装 CUDA 驱动

**解决方案：**
- 让用户使用 CPU 版本
- 或安装 NVIDIA 驱动

### Q3: 打包文件体积过大

**当前大小：**
- CPU版本：~150MB
- GPU版本：~200MB

**如果需要减小：**
1. 排除更多不需要的模块（编辑 spec 文件的 `excludes`）
2. 使用 UPX 压缩（`upx=True`，但会增加打包时间）

## 下一步

后端打包完成后，继续打包前端：

```bash
cd frontend

# 复制后端到资源目录
mkdir resources\backend
copy ..\backend\dist\xiaoyao-backend-cpu.exe resources\backend\

# 打包便携版
npm run electron:build:portable
```
