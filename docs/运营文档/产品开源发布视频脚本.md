## Remotion 实现要点

### 1. 动画驱动
所有动画必须使用 `useCurrentFrame()` 驱动，禁止使用 CSS 动画

### 2. 转场使用
使用 `@remotion/transitions` 的 `<TransitionSeries>` 组件

### 3. 文字动画
- 打字机效果：使用字符串切片
- 高亮效果：使用背景色渐变

### 4. 性能优化
- 图片资源使用 `staticFile()` 导入
- 大型静态内容使用 `<Still>` 组件
- 音频使用 `<Audio>` 组件并设置正确时长

### 5. 响应式设计
- 使用 `useVideoConfig()` 获取视频配置
- 相对尺寸使用 `%` 或 `interpolate()`

## AI配音方案

### 微软免费 TTS 使用方法

1. **在线版（推荐）**：
   - 使用 Edge 浏览器访问：https://speech.microsoft.com/tts
   - 选择语音：「晓晓」（自然女声）或「云健」（自然男声）
   - 输入配音文本，生成音频文件下载

2. **配音语音推荐**：
   - 女声：晓晓（自然、亲切）
   - 男声：云健（稳重、专业）
   - 语速：1.0x（正常）
   - 音调：默认

3. **配音流程**：
   ```
   场景1 → 生成音频1.mp3
   场景2 → 生成音频2.mp3
   ...
   场景6 → 生成音频6.mp3
   ```

4. **音频参数设置**：
   - 采样率：44.1kHz
   - 比特率：128kbps
   - 格式：MP3
