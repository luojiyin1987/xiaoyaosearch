> **视频类型**：产品宣传片

## 🎨 视觉风格规范

### 配色方案
```
主色：#2563EB (科技蓝) - 用于按钮、高亮
辅助色：#10B981 (成功绿) - 用于状态指示
强调色：#F59E0B (活力橙) - 用于重要提示
背景色：#0F172A (深空蓝) - 用于背景
文字色：#F8FAFC (亮白) - 用于主文字
```

### 字体规范
```
标题字体：思源黑体 Bold (中文) / Inter Bold (英文)
正文字体：思源黑体 Regular (中文) / Inter Regular (英文)
数字字体：SF Mono / JetBrains Mono (用于数据展示)
```

### 动画风格
- **转场**：流畅淡入淡出 + 轻微缩放
- **图标**：弹性缩放 + 路径动画
- **文字**：打字机效果 + 逐字浮现
- **UI**：原生界面动画（鼠标移动、点击反馈）

## 🎵 音频设计

### 背景音乐
- **风格**：科技感、轻快、现代
- **节奏**：120 BPM（中等节奏）
- **情绪**：
  - 开场（0-8s）：悬疑、探索感
  - 展示（8-35s）：轻快、愉悦
  - 技术（35-50s）：科技、专业
  - 结尾（50-60s）：激昂、号召性

### 音效清单
```
✅ Logo展示：科技感"叮"声
✅ 按钮/交互：轻微"咔嗒"声
✅ 搜索执行：快速"唰唰"声
✅ 结果展示：连续"嗒嗒"声
✅ AI处理：科技"嗡嗡"声
✅ 确认/成功：愉悦"叮咚"声
✅ 转场：轻微"嗖"声
```

### 配音要求
```
语调：专业、亲和、自信
语速：适中（约180字/分钟）
性别：男声/女声均可（建议30-40岁成熟声线）
语言：普通话（标准、清晰）
```
