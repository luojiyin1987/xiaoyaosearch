### ✅ 语雀 (yuque)

**状态**: 已内置（默认关闭） | **优先级**: 高 | **版本**: v1.0

**说明**:
- 系统内置插件，无需额外安装
- 默认关闭状态，需手动启用
- 使用 `yuque-dl` CLI 工具同步数据
- 自动下载 Markdown 格式文档

**特性**:
- 🔄 启动时自动同步（启用后）
- 📄 支持 Markdown 格式导出
- 🏷️ 保留文档元数据
- 🔗 支持源链接跳转

**启用方法**:

**步骤1**：将配置模板复制为实际配置文件
```bash
# 在插件目录下执行
cd backend/data/plugins/datasource/yuque
cp config.yaml.example config.yaml
```

**步骤2**：编辑 `config.yaml`，启用插件
```yaml
plugin:
  enabled: true  # 将 false 改为 true
```

**配置说明**:
配置模板文件：`backend/data/plugins/datasource/yuque/config.yaml.example`
实际配置文件：`backend/data/plugins/datasource/yuque/config.yaml`

完整配置示例（config.yaml）：
```yaml
plugin:
  id: yuque
  name: 语雀知识库
  version: "1.0.0"
  type: datasource
  enabled: true  # 手动启用

datasource:
  repos:
    - name: "个人品牌"                      # 知识库名称
      url: "https://www.yuque.com/username/repo/slug"  # 语雀知识库 URL
      download_dir: "./data/product-docs"  # 本地下载目录
      token: "your_yuque_token"            # 语雀 API Token
      ignore_images: true                  # 是否忽略图片下载
```

**使用说明**:
1. 获取语雀 API Token：[参考文档](https://github.com/gxr404/yuque-dl/blob/main/docs/GET_TOEKN.md)
2. 复制 `config.yaml.example` 为 `config.yaml`
3. 编辑 `config.yaml`，设置 `enabled: true`，配置知识库 URL 和 Token
4. 重启后端自动同步

**插件位置**: `backend/data/plugins/datasource/yuque/`

### ✅ 飞书 (feishu)

**状态**: 已内置（默认启用） | **优先级**: 高 | **版本**: v1.0

**说明**:
- 系统内置插件，无需额外安装
- 默认启用状态
- 零配置设计，自动识别飞书导出格式
- 支持从飞书导出的 Markdown 文档

**特性**:
- ✨ 零配置：自动识别飞书导出格式
- 🏷️ 元数据提取：从文档末尾提取原文链接
- ⚡ 性能优化：仅解析文件末尾500字符
- 🔗 支持源链接跳转

**使用说明**:
1. 从飞书导出文档为 Markdown 格式
2. 将导出的文件放置到扫描目录
3. 构建索引
4. 搜索结果将显示"飞书"标识

**飞书导出格式**:
```markdown
---
> 更新: 2026-03-30 02:52:46
> 来源类型: feishu
> 原文: <https://feishu.cn/wiki/MZKMwqpljiod1ak38Cscnr8hnkh>
---
```

**插件位置**: `backend/data/plugins/datasource/feishu/`
