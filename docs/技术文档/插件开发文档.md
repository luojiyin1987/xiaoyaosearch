## 插件系统概述

### 什么是插件？

小遥搜索的插件系统是一个模块化扩展框架，允许开发者通过独立插件扩展系统功能。每个插件都是独立的Python模块，可以安装、配置和使用，无需修改核心代码。

### 设计原则

**约定优于配置**：插件通过文件系统和配置文件管理，无需API接口。

- **安装方式**：将插件目录复制到 `data/plugins/` 下即完成安装
- **配置方式**：通过 `config.yaml` 文件配置
- **自动发现**：系统启动时自动扫描和加载插件
- **热插拔**：禁用插件只需重命名目录或修改配置

## 通用插件接口

### BasePlugin 基类

所有插件都必须继承 `BasePlugin` 基类，它提供了插件生命周期管理的基础功能：

```python
from abc import ABC, abstractmethod
from typing import Dict, Any

class BasePlugin(ABC):
    """插件基类，所有插件的父类"""

    # 插件元数据
    metadata: Dict[str, Any] = None

    @classmethod
    @abstractmethod
    def get_metadata(cls) -> Dict[str, Any]:
        """返回插件元数据

        Returns:
            Dict[str, Any]: 包含以下键的字典:
                - id: 插件唯一标识
                - name: 显示名称
                - version: 版本号（semver格式）
                - type: 插件类型（datasource/ai_model/search_engine等）
                - author: 作者
                - description: 描述
                - dependencies: 依赖列表（可选）
        """
        pass

    @abstractmethod
    async def initialize(self, config: Dict[str, Any]) -> bool:
        """初始化插件

        Args:
            config: 从config.yaml加载的配置字典

        Returns:
            bool: 初始化成功返回True，失败返回False
        """
        pass

    @abstractmethod
    async def cleanup(self):
        """清理资源（关闭连接、释放句柄等）"""
        pass

    def get_status(self) -> Dict[str, Any]:
        """获取插件运行状态（可选实现）"""
        return {
            "status": "running",
            "health": "healthy"
        }
```

### 插件元数据规范

所有插件的 `get_metadata()` 方法必须返回以下标准字段：

```python
{
    "id": "plugin_id",              # 插件唯一标识（必填）
    "name": "插件显示名称",           # 显示名称（必填）
    "version": "1.0.0",            # 版本号（必填，semver格式）
    "type": "datasource",          # 插件类型（必填）
    "author": "作者名",             # 作者（必填）
    "description": "插件描述",      # 描述（必填）
    "dependencies": [],            # 依赖的其他插件（可选）
    "min_system_version": "1.0.0"  # 最低系统版本要求（可选）
}
```

### 示例1：简单文件系统数据源插件

```python
# data/plugins/datasource/simple_file/plugin.py
import aiofiles
from pathlib import Path
from typing import AsyncIterator, Dict, Any
from app.plugins.interface import DataSourcePlugin

class SimpleFilePlugin(DataSourcePlugin):
    """简单文件系统插件"""

    @classmethod
    def get_metadata(cls) -> Dict[str, Any]:
        return {
            "id": "simple_file",
            "name": "简单文件系统",
            "version": "1.0.0",
            "type": "datasource",
            "author": "Your Name",
            "description": "扫描指定目录下的Markdown文件",
            "datasource_type": "simple_file"
        }

    async def initialize(self, config: Dict[str, Any]) -> bool:
        self.config = config
        self.scan_path = Path(config.get("scan_path", "."))
        if not self.scan_path.exists():
            raise ValueError(f"路径不存在: {self.scan_path}")
        return True

    async def scan(self, **kwargs) -> AsyncIterator[Dict[str, Any]]:
        for file_path in self.scan_path.rglob("*.md"):
            async with aiofiles.open(file_path, 'r', encoding='utf-8') as f:
                content = await f.read()

            yield {
                "id": f"simple_file_{file_path.stem}",
                "filename": file_path.name,
                "path": str(file_path),
                "content": content,
                "file_type": "markdown",
                "source_type": "simple_file",
                "source_url": None,
                "source_name": "简单文件系统"
            }

    async def cleanup(self):
        pass
```

### 示例2：OpenAI AI模型插件

```python
# data/plugins/ai_model/openai/plugin.py
import openai
from typing import AsyncIterator, Dict, Any
from app.plugins.interface import AIModelPlugin, ModelType

class OpenAIPlugin(AIModelPlugin):
    """OpenAI AI模型插件"""

    @classmethod
    def get_metadata(cls) -> Dict[str, Any]:
        return {
            "id": "openai",
            "name": "OpenAI",
            "version": "1.0.0",
            "type": "ai_model",
            "author": "XiaoyaoSearch Team",
            "description": "OpenAI GPT模型集成",
            "model_type": ModelType.TEXT.value,
            "supported_features": ["chat", "embedding", "streaming"]
        }

    async def initialize(self, config: Dict[str, Any]) -> bool:
        self.config = config
        self.client = openai.AsyncAI(api_key=config["api_key"])
        return True

    async def chat(self,
                   messages: list,
                   stream: bool = False,
                   **kwargs) -> AsyncIterator[Dict[str, Any]] | Dict[str, Any]:
        model = kwargs.get("model", self.config.get("model", "gpt-4"))

        if stream:
            stream_response = await self.client.chat.completions.create(
                model=model,
                messages=messages,
                stream=True
            )
            async for chunk in stream_response:
                if chunk.choices[0].delta.content:
                    yield {
                        "content": chunk.choices[0].delta.content,
                        "finish_reason": None
                    }
            yield {"finish_reason": "stop"}
        else:
            response = await self.client.chat.completions.create(
                model=model,
                messages=messages
            )
            return {
                "content": response.choices[0].message.content,
                "finish_reason": response.choices[0].finish_reason
            }

    async def embedding(self, text: str, **kwargs) -> list[float]:
        model = kwargs.get("model", self.config.get("embedding_model", "text-embedding-3-small"))
        response = await self.client.embeddings.create(
            model=model,
            input=text
        )
        return response.data[0].embedding

    async def cleanup(self):
        await self.client.close()
```

async def scan(self, **kwargs) -> AsyncIterator[Dict[str, Any]]:
    try:
        items = await self.client.get_items()
    except Exception as e:
        print(f"插件错误: {e}")
        return  # 静默失败，不影响其他插件

            yield self._process_item(item)
        except Exception as e:

            continue

## 常见问题

### Q1: 如何确定我的插件类型？

根据您要扩展的功能选择插件类型：

| 需求 | 插件类型 |
|------|----------|
| 添加新的数据源（如语雀、飞书） | 数据源插件 (datasource) |
| 集成新的AI服务（如Claude、Gemini） | AI模型插件 (ai_model) |
| 添加新的文件格式支持 | 内容解析器插件（规划中） |
| 替换搜索引擎 | 搜索引擎插件（规划中） |

### Q2: 插件之间可以通信吗？

不建议直接通信。插件应该保持独立，通过系统提供的服务交换数据：
- 使用数据库共享数据
- 使用系统事件机制
- 使用消息队列（如Redis）

### Q3: 如何测试插件？

```python
import pytest
from app.plugins.loader import PluginLoader

@pytest.mark.asyncio
async def test_my_plugin():
    loader = PluginLoader()
    plugin = await loader.load_plugin("my_plugin")
    assert await plugin.initialize({}) is True
    # 测试功能
    await plugin.cleanup()
```

### Q4: 插件可以依赖第三方库吗？

可以。在插件目录下创建 `requirements.txt`：

```txt
# data/plugins/my_plugin/requirements.txt
httpx==0.24.0
openai==1.0.0
```

### Q5: 如何禁用插件？

**方法1**：修改配置文件
```yaml
plugin:
  enabled: false
```

**方法2**：重命名插件目录
```bash
mv data/plugins/my_plugin data/plugins/my_plugin.disabled
```
